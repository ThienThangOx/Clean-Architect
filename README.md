# How to run
Clone this repository
```
git clone https://github.com/ThienThangOx/Clean-Architect.git
cd Fe
cd employees-management
```
(if you use npm)
```
npm install 
npm run serve 
```
(if you use yarn)
```
yarn install
yarn serve 
```
