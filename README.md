## FE

- Vuex
- I18n (Multiple laguage)

# How to run
Clone this repository
```
git clone https://github.com/ThienThangOx/Clean-Architect.git
cd Fe
cd employees-management
```
(For npm)
```
npm install 
npm run serve 
```
(For yarn)
```
yarn install
yarn serve 
```
